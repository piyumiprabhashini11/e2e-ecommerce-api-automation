## End to End e-commerce API Automation

Java + Rest Assured based end-to-end e-commerce API automation covering login, product, order, and delete flows.

## Tech Stack

- Java 11+
- Rest Assured
- Maven
- Hamcrest
- Jackson
- TestNG
- Eclipse IDE or IntelliJ
- Git & GitHub

## Setup Instructions

### 1. Clone the repository
```bash
git clone https://github.com/piyumiprabhashini11/e2e-ecommerce-api-automation.git
```
### 2. Prerequisites

Make sure the following tools are installed on your system:

Java 11+

Maven

Git

Eclipse IDE or IntelliJ IDEA

### 3. Verify installations:

```bash
java -version

mvn -version
```
